## 🧩 Cómo contribuir

Si quieres agregar una nueva definición o corregir algo:
1. Haz un **fork** del repositorio.
2. Crea un nuevo archivo Markdown dentro de la carpeta `definiciones/`.
3. Envía un **pull request** con tus cambios.
